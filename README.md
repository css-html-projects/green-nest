## Features

- Advantages section highlighting delivery, returns, support, and secure payment
- Indoor and outdoor plant showcases
- Catalog with product cards, prices, and order buttons
- Newsletter subscription form
- Blog section with recent articles
- Footer with site map and legal links
- Responsive burger menu for mobile devices
